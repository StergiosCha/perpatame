# Logo Assets

## Required Files

Please place the following logo files in this directory:

1. `ms-federation-logo.png` - The MS Federation logo
2. `simasia-logo.png` - The Simasia AI logo

## Specifications

- **Format**: PNG (with transparent background recommended)
- **Recommended height**: 120-180px (will be displayed at 60px height, so 2x for retina displays)
- **Aspect ratio**: Maintain original logo proportions

## Usage

These logos are displayed in the header of all three frontend pages:
- Display page (`frontend/display/index.html`)
- Moderate page (`frontend/moderate/index.html`)
- Submit page (`frontend/submit/index.html`)

The logos will appear horizontally aligned at the top of each page with a subtle hover effect.
